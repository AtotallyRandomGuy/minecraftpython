# MCPY
Minecraft, but made in Python 3 and pyglet.
Open to pull requests


To run, install pyglet and Python 3.
Go the the project's directory and open a terminal or command prompt.

# macOS and Linux
python3 minecraft.py

# Windows
py minecraft.py
